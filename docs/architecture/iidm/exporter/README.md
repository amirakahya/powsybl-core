# IIDM - Exporters

The `com.powsybl.iidm.export.Exporter` class is used to generate a text file in a given format describing a network
from an IIDM network instance. Powsybl supports natively several exporters:
- [IIDM](iidm.md)
- [AMPL](ampl.md)
